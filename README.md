# Cat Energy

Адаптивный сайт для сервиса функционального питания домашних животных. Проект выполнен по методологии БЭМ с использованием подхода Mobile First.

## Стек

* HTML
* SCSS
* JavaScript
* Gulp

## Возможности

* полностью адаптивная вёрстка;
* Mobile First;
* мобильное меню;
* интерактивная карта;
* формы;
* семантичная разметка;
* интерактивные состояния элементов.

## Особенности

Во время разработки особое внимание уделялось:

* построению адаптивной сетки;
* Pixel Perfect-вёрстке нескольких макетов;
* организации масштабируемой структуры проекта;
* использованию современных возможностей CSS.

## Запуск

npm install
npm start
